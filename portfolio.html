Giving

Giving is a full-stack web application linking donors of material goods to local charitable institutions in need of those items. This application was built by a team of developers using MongoDB, Express, React, Node, HTML5, CSS3, JavaScript, Bootstrap, and Passport for user authentication.

See the GitHub Repository for this application.

Title Dive

Title Dive is a responsive web application providing specific movie and television show recommendations based on user title input. Users receive title recommendations, together with viewing data and movie trailers to ensure viewer compatibility. This application was built by a team using HTML5, CSS3, JavaScript, Bootstrap, jQuery, AJAX and APIs.

See the GitHub Repository for this application.
